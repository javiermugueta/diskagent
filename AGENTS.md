# AGENTS

Input principal para agentes “desde cero”:
- `PRODUCT_SPEC.md`

Uso recomendado:
1. Entregar `PRODUCT_SPEC.md` al agente como instrucción base.
2. Pedir implementación completa con código, tests, empaquetado y documentación.
3. Validar con los comandos definidos en la propia especificación.

Este archivo solo existe para señalar de forma explícita cuál es el documento de entrada canónico.
